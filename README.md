# App Finanzas SAG

Monitor de inversiones financieras con datos en tiempo real de Argentina y el mundo.

## Características

- 🌍 **Monitor Global**: Índices bursátiles internacionales (S&P 500, Nasdaq, Dow Jones, etc.)
- 🇦🇷 **Argentina**: MERVAL, acciones argentinas, dólar oficial/MEP/CCL, bonos soberanos
- 📊 **Datos en tiempo real**: APIs de data912, Bluelytics, ArgentinaDatos, BCRA
- 📱 **Responsive**: Funciona en desktop y móvil
- 🎨 **Interfaz moderna**: Tema oscuro con animaciones

## Tecnologías

- HTML5, CSS3, JavaScript (ES6+)
- Chart.js para gráficos
- APIs públicas argentinas

## Cómo usar

1. Abre `index.html` en tu navegador
2. Navega entre las pestañas Mundo y Argentina
3. Los datos se actualizan automáticamente cada 60-300 segundos
