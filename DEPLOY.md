# 🚀 Cómo publicar App Finanzas SAG en GitHub Pages

## Paso 1: Crear cuenta en GitHub
Si no tienes cuenta, ve a https://github.com y regístrate (gratis).

## Paso 2: Crear nuevo repositorio
1. Click en "New repository"
2. Nombre: `app-finanzas-sag` (o el que prefieras)
3. Descripción: "Monitor de inversiones financieras Argentina"
4. **IMPORTANTE**: Marca "Add a README file" como NO
5. Click "Create repository"

## Paso 3: Subir archivos
### Opción A: GitHub Desktop (Recomendado)
1. Descarga GitHub Desktop: https://desktop.github.com
2. Abre GitHub Desktop
3. File > Clone repository > Tu repositorio nuevo
4. Copia todos los archivos de esta carpeta al directorio clonado
5. En GitHub Desktop: Commit y Push

### Opción B: Manual en GitHub
1. En tu repositorio, click "uploading an existing file"
2. Arrastra todos los archivos de esta carpeta
3. Click "Commit changes"

## Paso 4: Activar GitHub Pages
1. Ve a tu repositorio en GitHub
2. Click en "Settings" (engranaje)
3. En el menú izquierdo, click "Pages"
4. En "Source", selecciona "Deploy from a branch"
5. Branch: "main", folder: "/ (root)"
6. Click "Save"

## Paso 5: ¡Listo!
Después de unos minutos, tu sitio estará disponible en:
`https://[tu-usuario].github.io/[nombre-repositorio]/`

**Ejemplo:** `https://johndoe.github.io/app-finanzas-sag/`

## Compartir
- Comparte la URL con quien quieras
- Se puede ver desde cualquier PC, teléfono o tablet con internet
- Los datos se actualizan en tiempo real

## Actualizaciones
Para actualizar el sitio:
1. Modifica los archivos localmente
2. Sube los cambios a GitHub
3. GitHub Pages se actualiza automáticamente en minutos

---
**¿Problemas?** Revisa que todos los archivos se subieron correctamente y que GitHub Pages esté activado.
