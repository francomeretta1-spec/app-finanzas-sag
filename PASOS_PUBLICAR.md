# 📋 PASO A PASO: Publicar App Finanzas SAG

## PASO 1: Crear cuenta GitHub (2 minutos)
1. Ve a https://github.com
2. Click "Sign up" (arriba derecha)
3. Elige usuario (ej: francgomez)
4. Email y contraseña
5. Verifica tu email
6. ¡Listo! Ya tienes cuenta gratis

## PASO 2: Crear repositorio (1 minuto)
1. En GitHub, click el botón verde "+" → "New repository"
2. Nombre: `app-finanzas-sag`
3. Descripción: "App de finanzas Argentina"
4. **IMPORTANTE**: NO marques "Add a README file"
5. Click "Create repository"

## PASO 3: Subir archivos (3 minutos)
1. En tu repositorio nuevo, click "uploading an existing file"
2. Arrastra TODOS los archivos de tu carpeta `app-finanzas`:
   - index.html
   - script.js
   - style.css
   - README.md
   - .gitignore
   - .nojekyll
   - DEPLOY.md
3. Escribe mensaje: "Subida inicial de App Finanzas"
4. Click "Commit changes"

## PASO 4: Activar sitio web (1 minuto)
1. En tu repositorio, click "Settings" (engranaje)
2. En menú izquierdo, click "Pages"
3. En "Source": selecciona "Deploy from a branch"
4. Branch: "main" | Folder: "/ (root)"
5. Click "Save"

## PASO 5: ¡OBTENER TU LINK! (espera 2-3 minutos)
1. Espera 2-3 minutos
2. Ve otra vez a Settings → Pages
3. Verás tu link: `https://TU-USUARIO.github.io/app-finanzas-sag/`
4. Click el link para probar

## ✅ RESULTADO FINAL
- Link público: `https://tu-usuario.github.io/app-finanzas-sag/`
- Cualquiera puede usarlo desde cualquier dispositivo
- Datos se actualizan solos cada 5 minutos

## 🔄 PARA ACTUALIZAR DESPUÉS
1. Modifica archivos en tu PC
2. Ve a repositorio GitHub
3. Click "Add file" → "Upload files"
4. Sube archivos modificados
5. Se actualiza automáticamente

---
**¿Problema?** Asegúrate de que subiste TODOS los archivos y que Pages esté activado.
